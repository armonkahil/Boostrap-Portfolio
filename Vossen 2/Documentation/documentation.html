General Information

This template includes following pages:

  • index.html
  • home-video.html
  • home-fullwidth.html
  • home-landing-page.html
  • shortcodes.html

Stylesheets (CSS files) are placed in /css/ folder.
All Images are in /img/ folder.
Javascripts files are in /js/ folder.
And all PHP files are in /php/ folder.

HTML Structure

The template is based on latest Bootstrap 3, the most popular framework for developing responsive, mobile friendly projects.

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options, as well as powerful mixins for generating more semantic layouts.

Create a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding, then create .row and add columns with appropriate device name (xs, sm, md, lg) and grid number (2, 4, 6, 12).

For full documentation, please visit Bootstrap 3 site: http://getbootstrap.com/css/ 

<div class="container">
<div class="row">
    <div class="col-lg-12">
      ...
    </div>
</div>
</div>

Google Map

Embed a Google map

  1. Open Google Maps.
  2. Go to the directions, map, or Street View image you'd like to embed.
  3. In the top left, click Menu
  4. Click Share or embed map.
  5. Click Embed map.
  6. Copy the text in the box. Paste it into vossen-map section of your html file.

<!-- Map -->
<section class="vossen-map">

<!-- Paste your map iframe here -->
<iframe height="550" src="https://maps.google.com/maps?q=Renaissance%20New%20York%20Hotel%2057%2C%20New%20York%2C%20USA&t=m&z=17&output=embed&iwloc=near" border="0" marginwidth="0" marginheight="0"></iframe>

</section>
<!-- End Map -->

Twitter Feed

jQuery Tweetie - Simple Twitter Feed Script that works with new Twitter 1.1 API.

One notice: Twitter Feed will be functional only on your server, php files can not be executed on your local computer.
Upload to your server when testing.


New Twitter API requires oAuth Token Key, so it's three step process.

Step 1

First, you need a consumer key and secret keys. Get one from https://dev.twitter.com/apps

Step 2

In "/ php /" folder edit "twitter-info.php" file and replace variables with your Consumer and oAuth Keys.


config.php: 

// Consumer Key
define('CONSUMER_KEY', 'CONSUMER_KEY_HERE');
define('CONSUMER_SECRET', 'CONSUMER_SECRET_HERE');

// User Access Token
define('ACCESS_TOKEN', 'ACCESS_TOKEN_HERE');
define('ACCESS_SECRET', 'ACCESS_SECRET_HERE');

Step 3

Open scripts.js file from js folder and on line 136 put in your username between quotation marks: 
/* -------------------
Twitter Feed
---------------------*/
$('.tweet').twittie({
username: 'your-username',
dateFormat: '%b. %d, %Y',
template: '{{tweet}} <div class="date">{{date}}</div>',
count: 3,
hideReplies: false
});

Subscribe Mailchmp

The subscription form is integrated with Mailchimp mailing list.
You need to have an active Mailchimp account and to create a list.

Once you have the Mailchimp account and list you have to configure the subscription file with your API Key & List ID.
In / php / folder open the subscribe-mailchimp.php file and and change the variables:

subscribe-mailchimp.php :

// Authentication
$api_key    = 'your-api-key';
$list_id    = 'your-list-id';

- How to find or generate your API Key * http://kb.mailchimp.com/accounts/management/about-api-keys
- How to find your List ID * http://kb.mailchimp.com/lists/managing-subscribers/find-your-list-id

One notice: Subscription Form will be functional only on your server, php files can not be executed on your local computer.
Upload to your server when testing.
